Introduction

Introduction

Welcome to Moon Rabbit docs. Here you can find all information about the Moon Rabbit network.

Note: system is under active development, so we will extend docs while developing

System overview

Moon Rabbit enables scalability by allowing specialized blockchains to communicate with each other in a secure, trust-free environment.

Moon Rabbit is built to connect and secure unique blockchains, whether they be public, permissionless networks, private consortium chains, or oracles and other Web3 technologies. It enables an internet where independent blockchains can exchange information under common security guarantees. These blockchains are called Jurisdictions.

Networks

There are two networks currently running which are mainnet and testnet.

You can connect to these networks using the following public nodes:

Mainnet - wss://main.hole.moonrabbit.com

Testnet - wss://node.hole.moonrabbit.com
